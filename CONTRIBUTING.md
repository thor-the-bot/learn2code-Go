# Contributing

Thank you for your interest in contributing to learn2code!

This repository requires contributors to sign a Contributor License Agreement (CLA) before accepting contributions.

How to contribute
1. Fork the repository.
2. Implement your change in a feature branch.
3. Open a pull request describing the change.

CLA
Before we can merge your contribution, please add a signed CLA using the TEMPLATE-CLA.md provided in this repo. The CLA is a simple agreement that grants the project the right to accept and relicense contributions while protecting contributors' rights.

Editing exercises
- Keep solutions out of the `exercises/` folder. Place any reference solutions in `solutions/` and ensure `solutions/` is in `.gitignore`.
- Each exercise should include a short README describing the learning objective and a set of tests that define expected behaviour.

Code style
- Follow Go idioms and `gofmt` formatting.
- Keep functions small and focused.
